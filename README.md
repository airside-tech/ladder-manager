# Cable ladder manager

The manager is a tool to plan and route cable ladders inside data centers /equipment rooms.
All equipment rooms are assumed defined as a room with a datafloor given by the matrix of 60 x 60 cm tiles.
The back-end is written in Python. Front-end framework is not decided.
